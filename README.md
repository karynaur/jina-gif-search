
# Jina gif search


## Run

| Command | Description |
| :--- | :--- |
| ``python app.py index`` | To index files/data |
| ``python app.py search`` | To run query on the index |
| ``python app.py dryrun`` | Sanity check on the topology |

## View in Browser

```bash
cd static
python -m http.server
```

Open `http://0.0.0.0:8000/` in your browser.
